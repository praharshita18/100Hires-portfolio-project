# 100Hires Portfolio Project

## Overview

This repository was created as part of the 100Hires portfolio project. The objective was to install the required AI development tools, create a GitHub repository, open it in Cursor, and document the process.

## Tools Installed

* Cursor IDE
* Claude Code Extension for Cursor
* Codex Extension for Cursor
* GitHub (existing account)

## Steps Completed

1. Installed Cursor IDE.
2. Installed the Claude Code extension in Cursor.
3. Logged into Claude Code using a Claude account.
4. Installed the Codex extension in Cursor.
5. Logged into Codex using an OpenAI account.
6. Created a public GitHub repository.
7. Opened the repository in Cursor.
8. Created this README file to document the setup process.

## Challenges Encountered

### Claude Code Authentication

After installing the Claude Code extension, it was not immediately obvious that authentication was required. I explored the extension interface, located the login options, and successfully authenticated using my Claude account.

### Codex Authentication

Similar to Claude Code, Codex required account authentication before use. I connected my OpenAI account and verified access.

### Repository Setup

With a Computer Science Engineering background and prior experience creating and managing GitHub repositories through my personal portfolio projects, setting up repositories was straightforward. The main task was becoming familiar with Cursor, as I am more accustomed to working with Claude Code. I connected the repository to Cursor, explored its workflow, and confirmed that the development environment was configured correctly.

## Key Learnings

* How to install and configure AI-powered development tools in Cursor.
* How to connect external AI services through authentication.
* How to create and manage GitHub repositories.
* How to document technical processes clearly and concisely.
